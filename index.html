Cloud Native development

SmallRye improves the developer experience for Cloud Native development through implementing Eclipse MicroProfile. Offering important functionality for Cloud environments.

Community implementations

SmallRye develops implementations of Eclipse MicroProfile that are usable by any vendor or project. Come join the community to contribute to shared implementations of the specifications.

Containers are great!

With containers being the new operating system, it's important that developers are able to build applications that can easily be deployed to container environments.
